*** Kurinto Font Folio v2.197 [Lite] - Map

This file provides a roster of the source fonts and writing systems that make
up each Kurinto font. It shows which specific source files are included in each
Kurinto font, and which writing systems (scripts) they contribute.

The writing systems are identified by script codes that are listed in the
"Script Codes" section of the Technical Information part of the Kurinto User's
Guide. These script codes are specific to Kurinto and, while they are related
to the script codes used by Unicode, there are significant differences.

A number of the fonts listed are internal to the Kurinto development and
pipeline process. In particular, the Kurinto Prime font, Egypt Format Controls,
and any font name containing "CG" are described in the Internal Fonts section
following the Source Map.

*** Source Map

The Kurinto fonts in this list are in alphabetical order.
